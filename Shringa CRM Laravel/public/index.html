Laravel Application Not Loading
The Laravel application is not properly configured for Vercel deployment.
Troubleshooting:
1. Check if all environment variables are set in Vercel dashboard
2. Ensure APP_KEY is generated and set
3. Database connection should be configured
4. Check Vercel function logs for errors

If you're seeing this page, it means the Vercel deployment is working but Laravel is not loading properly.

Expected URL: The Laravel application should load at the root URL.
